# Generate Parentheses

## Problem Statement

Given n pairs of parentheses, write a function to _generate all combinations of well-formed parentheses._

---

**Example 1**

**Input**: n = 3

**Output**: ["((()))","(()())","(())()","()(())","()()()"]

---

**Example 2**

**Input:** n = 1

**Output:** ["()"]
